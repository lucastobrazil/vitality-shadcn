## API Reference

### PasswordInput

Accepts all standard `<input>` props except `type`, which is managed internally.

| Prop          | Type     | Default | Description        |
| ------------- | -------- | ------- | ------------------ |
| `className`   | `string` | -       | Additional classes |
| `placeholder` | `string` | -       | Placeholder text   |

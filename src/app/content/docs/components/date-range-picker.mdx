## Usage

```tsx

```

```tsx
<DateRangePicker />
```

### Controlled

```tsx
import type { DateRange } from "react-day-picker"

const [range, setRange] = useState<DateRange | undefined>()

<DateRangePicker value={range} onChange={setRange} />
```

## API Reference

### DateRangePicker

| Prop             | Type                                        | Default              | Description                            |
| ---------------- | ------------------------------------------- | -------------------- | -------------------------------------- |
| `value`          | `DateRange \| undefined`                    | -                    | Controlled selected date range         |
| `onChange`       | `(range: DateRange \| undefined) => void`   | -                    | Callback when range changes            |
| `placeholder`    | `string`                                    | `"Pick a date range"`| Text shown when no range is selected   |
| `numberOfMonths` | `number`                                    | `2`                  | Number of months to display            |
| `className`      | `string`                                    | -                    | Additional classes for the trigger     |

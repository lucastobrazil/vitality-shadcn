## Usage

```tsx
import { DatePicker } from "@/registry/vitality/ui/date-picker"
```

```tsx
<DatePicker />
```

### Controlled

```tsx
const [date, setDate] = useState<Date | undefined>()

<DatePicker value={date} onChange={setDate} />
```

## API Reference

### DatePicker

| Prop          | Type                                | Default         | Description                         |
| ------------- | ----------------------------------- | --------------- | ----------------------------------- |
| `value`       | `Date \| undefined`                 | -               | Controlled selected date            |
| `onChange`    | `(date: Date \| undefined) => void` | -               | Callback when date changes          |
| `placeholder` | `string`                            | `"Pick a date"` | Text shown when no date is selected |
| `className`   | `string`                            | -               | Additional classes for the trigger  |

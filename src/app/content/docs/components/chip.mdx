## API Reference

### Chip

| Prop        | Type      | Default | Description                           |
| ----------- | --------- | ------- | ------------------------------------- |
| `asChild`   | `boolean` | `false` | Render as child element (e.g. a link) |
| `className` | `string`  | -       | Additional classes                    |

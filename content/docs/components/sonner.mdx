## About

Sonner is built and maintained by [emilkowalski](https://twitter.com/emilkowalski).

## Installation

<InstallCommand name="sonner" />

Add the `Toaster` component to your root layout:

```tsx title="app/layout.tsx"
import { Toaster } from "@/registry/vitality/ui/sonner"

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <head />
      <body>
        <main>{children}</main>
        <Toaster />
      </body>
    </html>
  )
}
```

## Usage

```tsx
import { toast } from "sonner"
```

```tsx
toast("Event has been created.")
```

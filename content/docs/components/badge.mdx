## Usage

```tsx

```

```tsx
<Badge>Default</Badge>
<Badge variant="neutral">Neutral</Badge>
```

## API Reference

| Prop      | Type                              | Default     | Description                        |
| --------- | --------------------------------- | ----------- | ---------------------------------- |
| `variant` | `"default" \| "neutral"`          | `"default"` | The visual style of the badge.     |
| `asChild`  | `boolean`                         | `false`     | Render as a child component.       |
